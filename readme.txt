== Description ==

Adds "#FINDELACITA" to the end of every page. This plugin does nothing else.

== Installation ==

1. Upload `findelacita.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
